MASTER Document Index  (90:10 Open Access Model)

90:10 Open Access Model -- Guidance  ( HERE )

Document to define the procedure for achieving '90:10 Open Access' to Strictly Confidential Data, including the key concepts, role accountabilities, and process steps.

Open Access Collection (OAC) Agreement  ( HERE )

Document to define the Open Access Collection; it does not grant access to the data collection but instead defines the criteria that specify the scope

Data Use Terms (AOT)  ( HERE )

Document to formalise the agreement between (Delegate) Data Owners and Data Consumers - to determine the specified data-scope, user-base and terms of use 

User Guide  ( HERE )

Guide to explain what the '90:10 Open Access Model' is, and the required steps in the User Journey to enable access -- including links to necessary training, and other helpful resources

Collection Manager Solution  (HERE)

Live view to show which studies (Study Manager) & users (People Manager) are in scope for available data collections -- maintained by R&D Data Office

Comms & Change Materials  (HERE)

Master document with key comms materials & rollout plan agreed by Delegate Data Owners and stakeholders -- including details on what is changing for those involved -- and who is getting  what,by  when
