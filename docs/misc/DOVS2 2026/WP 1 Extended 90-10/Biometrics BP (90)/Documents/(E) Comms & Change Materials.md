Context

In 2025, the R&D Data Office will streamline data sharing and access to increase the speed and scale of data available across R&D, maximizing impact while actively managing risk.

Two changes will drive this shift: Primary Use reinterpretation replacing Extended Primary Use (EPU), and a new Role-based Open Access model (ROAM) for strictly confidential clinical trial data, enabling qualified open access for approved users.

Phase 1 began in Q3-Q4 2025, where we started co-shaping ROAM with TA stakeholders, and shared comms around the new policy improvements; Phase 2 (2026) will begin scaled rollout of ROAM to key groups -- incrementally expanding tech, tools, and broader data scopes.

Key Changes

Primary Use Reinterpretation:  Following the June '25 DGB decision, we are retiring EPU, and full Primary Use now governs data use within Drug Development (per CSP). This expands available patient data, removes time‑consuming sub‑setting - and delivers broader, faster access to clinical datasets within consent & privacy safeguards.

Role-based Open Access Model (ROAM):  We are shifting from project-by-project approvals to role-based open access through umbrella approvals -- targeting that 90% of user needs are open by default for qualified use -- rolling out in phases across 2025 -- 2026+ as policy, proactive data preparation, and technology mature.

Role-based Open Access Model (ROAM)

How will it work?

9

Role-based Open Access Model (ROAM) = New role‑based, open access model to pre‑approved data collections -- approved by Delegate Data Owners and accessible in available R&D platforms

Primary Use Reinterpretation:   Leveraging reinterpretation of Primary Use to streamline access & reduce need for sub-setting -- operating under existing Internal Sharing Standards

Open Access Collections:   Studies and data grouped into pre‑approved collections, defined and governed by Delegate Data Owners to enable controlled open access.

Data Use Terms:   Approves which user groups can access collections, for role‑aligned intent under permitted use -- continuous access approved top‑down by Delegate Data Owners

Access Solutions & Environments:   Leverages R&D's approved analysis platforms (eg: Domino), in combination with innovative new access mechanisms and tools (eg: Immuta)

Collection Management:   Joint stewardship by Data Office and TA's -- to update studies, add new users, and adjust access to keep collections current and controlled
