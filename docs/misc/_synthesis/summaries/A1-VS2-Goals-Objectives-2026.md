# Summary: VS2 Goals & Objectives - 2026

**Source:** `DOVS2 2026/General/VS2 Goals & Objectives - 2026.md`
**Type:** Strategy/Goals Document
**Workstream:** All (VS2 Program Overview)
**Date:** 2026

---

## Key Points

- **Overall Vision:** Embed AI-enabled, AI-ready data sharing paradigm across R&D `[ref: line 8-14]`
- **Scale targets:** 10 groups, >1,000 users; >1,700 studies AI-ready `[ref: line 10-13]`
- **Two key tools:** Collectoid (data access management) and Sherlock (AI data discovery) `[ref: line 13-14]`
- **"90:10 Principle":** 90% of user data needs via open access models, 10% via streamlined requests `[ref: line 18-22]`

---

## Entities/Concepts Defined

- **ROAM (Role-based Open Access Model):** Open access model for data sharing `[ref: line 24-29]`
- **Broad Build:** One of the open access models `[ref: line 21]`
- **Data Product Access:** One of the open access models `[ref: line 21]`
- **IDAP:** Streamlined request process for the "10%" `[ref: line 22]`
- **Collectoid:** "90 Data Collections Manager" - tool for managing open access `[ref: line 132-133]`
- **Sherlock:** "AI-enabled Find & Explore" - tool for data discovery `[ref: line 142]`
- **DPO:** Data Protection Office (manages collections) `[ref: line 43-46]`
- **BIC:** Best in Class standards `[ref: line 44]`

## Workstream Goals

### WP1 - 90:10 Data Access `[ref: lines 16-46]`
- **Phase 1 (Q2):** ROAM rollout to Oncology and Biopharma - 5 groups (ODSAI, Early Oncology, Biometrics BP, Biometrics ONC, EAI), >500 users
- **Phase 2 (Q4):** Additional 5 groups (Patient Safety/TRISARC, TSEM, CPSS, Clinical Operations, Late Clinical), >500 more users
- **Maturity:** Q1 publish ROAM v1; Q2 hand-over to DPO; Q3 ROAM v2; Q4 transition to BAU

### WP2 - R&D Data Exchange `[ref: lines 52-75]`
- **Q2:** Agree "north star" for external sharing aligned to AI-readiness and 2030 ambition
- **Q4:** Deliver 5 measurable process improvements
- **Status:** Goals incomplete (placeholder text) - vision still being defined

### WP3 - Data Availability `[ref: lines 77-119]`
- Make >1,700 "90-able" studies AI-ready
- Cover all SDTM, ADAM, imaging, omics, consent, samples, study metadata
- Ensure new clinical data automatically flows with standardised semantics/metadata/governance
- **Status:** Goals incomplete (placeholder text)

### WP4 - Agentic AI & Tech `[ref: lines 121-150]`
- **Q2:** Launch initial Sherlock for AI-assisted data discovery
- **Q4:** Collectoid actively supporting 90:10 with productivity gains
- **Goals:** Auditability and monitoring embedded; support the '10' functionality
- **Status:** Goals incomplete (placeholder text)
