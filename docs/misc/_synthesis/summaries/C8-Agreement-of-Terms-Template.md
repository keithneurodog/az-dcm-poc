## Requirements `[ref: Section 2.3.1]`

### Request Review (Due Diligence):
- **Scientific Merit:** R&D Data Office documents single RAP for this Agreement
- **Contractual Restrictions:** Assessment of divestments, out/in-licensing before access
- **Data Availability:** Cannot share if AZ doesn't have access
- **Legal/Ethical Basis:** Primary Use only; must align with Protocol/ICF; privacy laws (GDPR, HIPAA); HGR for China data
- **Business Sensitivity:** Closed studies >6mo post-DBL eligible by default; exceptions flagged by GPTs/DDOs

### Data Protection Principles:
- **Data Sub-setting:** Not required for Primary Use
- **Secure Environment:** Password-protected, security monitored
- **Legal Protection:** Code of Ethics, Data Privacy Standard obligations (a-i listed)

### Share Insights:
- Analysis outcomes made available to AZ community
- Product Team vetting required before decisions/publications
- Publication Sign-off (PSO) process applies

### GxP Requirements:
- Analyses for decision-making/submissions must be in GxP environment
- Outputs stored in GxP environment
