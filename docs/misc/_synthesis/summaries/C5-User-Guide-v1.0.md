## Access Mechanisms (v1.0) `[ref: Section 3.5]`

### 1. Data Product Solution (PDP)
Same as v2.0:
- SPDP pipeline with SDTM standard data in Starburst
- Environments: SCP, Domino, AI Bench, entimICE
- Query on demand, no bulk copy

### 2. Golden Ticket Solution (REMOVED in v2.0)
- **Purpose:** Access original study data with pre-approved permissions
- **Use case:** SAS files in entimICE, imaging/omics in IO Platform, Genestack, QuartzBio
- **Duration:** Open for duration of study/collection
- **Note:** Marked `[Pending review with team]` and `[Details need to be aligned -- action pending 26/11/2025]`

## User Journey (same as v2.0) `[ref: Section 1.1]`

One-time setup:
1. Awareness - Contact email, Knowledge Hub
2. Compliance - Training, access setup

Per-project:
1. Ideation - 90:10 Power BI Dashboard
2. Pre-planning - Self-check against User Guide
3. Execution - PDP or Golden Ticket
4. Close-down - Project Completion Form

## Primary Use vs Secondary Use (same as v2.0) `[ref: Section 2.2]`

**Primary Use (Permitted):**
- R&D activities supporting AZ drug development
- AI analytics (excluding development/training)

**Secondary Use (Not Permitted):**
- AI development/training
- External data sharing
